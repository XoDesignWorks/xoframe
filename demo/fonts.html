XOframe

web font stability

Fonts are the #2 cause of CLS

When a web font swaps in with different metrics than the fallback, every line of text reflows. @xodesign/xoframe/fonts registers a metric-adjusted fallback @font-face (measured at runtime, or from precomputed metrics) so the fallback occupies the same space as the web font — the swap shifts nothing. It also preloads fonts and adds .xo-fonts-loaded to <html> for a controlled reveal.
