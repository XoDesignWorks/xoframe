XOcarousel — horizontal & vertical

The browser does the hard part

Native CSS scroll-snap gives momentum, touch and trackpad scrolling for free — no animation loop, no transform math, no scroll listener. JS only adds arrows, dots, autoplay, mouse drag and active tracking (via IntersectionObserver, so INP stays low). Because the scrolling itself is CSS, you can link xoframe-carousel.css and pre-render the track to get a working snapping strip even with JS disabled — JS then only upgrades it with arrows, dots and autoplay.

Horizontal, autoplay + pause control

Autoplay is off by default (the accessible default). When enabled it ships a real pause/play button, pauses on hover and focus, and once you press pause it stays paused — as WCAG 2.2.2 requires. Drag it with the mouse.

Vertical axis

Same engine, axis: 'y' — snapping on the Y axis, arrows on top and bottom, dots down the side. Use ↑/↓.
